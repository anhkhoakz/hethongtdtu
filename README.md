# TDTU Student Portal

A fast, minimal, and practical web portal for convenient access to the TDTU Student Portal and related services.

## Features

- **Lightning fast**: Pure HTML and CSS with minimal JavaScript, no frameworks or build tools.
- **Minimalist design**: Clean, modern, and distraction-free interface.
- **Mobile-friendly**: Fully responsive and works on all devices.
- **Dark mode**: Automatic dark mode support with manual toggle, respects system preferences.
- **PWA ready**: Installable as a progressive web app on mobile devices.
- **Accessibility**: ARIA labels, semantic HTML, and keyboard navigation support.
- **Direct links**: Quickly access all essential TDTU student resources.
- **Section anchors**: Jump to any section using URL fragments (e.g., `#education`, `#login`).
- **Quick navigation**: Sticky navigation bar for easy section jumping.
- **SEO optimized**: Comprehensive meta tags for better search engine visibility and social sharing.

## Usage

Just open [the deployment site](https://anhkhoakz.github.io/hethongtdtu/) in your browser. Use URL fragments to jump to sections, e.g.:

- `/#login`
- `/#education`
- `/#learning-management-system`
- `/#student-affairs`
- `/#tuition-service-charge`
- `/#inspire-library`

### Installing as PWA

On mobile devices:

1. Open the site in your browser
2. Tap the "Add to Home Screen" option
3. The portal will be installed as an app icon

### Dark Mode

Click the 🌓 button in the footer to toggle between light and dark modes. The site will remember your preference.
